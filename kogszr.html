WINGS

  • 숲의 창조자의 날개 (Dragon by Wildkin) + 포세이돈의 권속 (Dragon next to Rexxar) = 물의 창조자의날개
  • 물의 창조자의날개 (Previous) + 백화조의 깃털 (Owl in 2nd city) = 백화조의 날개
  • 백화조의 날개 (Previous) + 노아의 깃털 (Angel guy in chimera cave) = 추방자의 날개
  • 추방자의 날개 (Previous) + 마왕의 날개가죽 (Devil (Dragon human)) = 마왕의 날개

WEAPON

  • 용사의 대검 (Death knight satyr dung) + 불독의 총 (2nd Chimera tower boss) = 불독의 제련대검
  • 불독의 제련대검 (Previous) + 아다만티움 (7th Chimera tower boss) = 아다만티움 대검
  • 용사의 단검 (Death knight satyr dung) + 불독의 총 (2nd Chimera tower boss) = 불독의 제련단검
  • 불독의 제련단검 (Previous) + 아다만티움 (7th Chimera tower boss) = 아다만티움 단검
  • 용사의 지팡이 (Death knight satyr dung) + 불독의 총 (2nd Chimera tower boss) = 불독의 제련완드
  • 불독의 제련완드 (Previous) + 아다만티움 (7th Chimera tower boss) = 아다만티움 완드

ARMOR

  • 현자의 로브 (Antondias satyr dung) + 스카디우스의 뼈 (4th Chimera tower boss) = 스카디우스 갑옷
  • 스카디우스 갑옷 (Previous) + 불카노스의 심장 (8th Chimera tower boss) = 불카노스 갑옷
  • 현자의 로브 (Antondias satyr dung) + 스카디우스의 날개가죽 (4th Chimera tower boss) = 스카디우스 가죽옷
  • 스카디우스 가죽옷 (Previous) + 불카노스의 심장 (8th Chimera tower boss) = 불카노스 가죽옷
  • 현자의 로브 (Antondias satyr dung) + 스카디우스의 옷천 (4th Chimera tower boss) = 스카디우스 로브
  • 스카디우스 로브 (Previous) + 불카노스의 심장 (8th Chimera tower boss) = 불카노스 로브

RING

  • 타이젠의 뿔반지 (Revenant satyr dung) + 불카노스의 불 (3rd Chimera tower boss) = 불카노스의 반지
  • 불카노스의 반지 (previous) + 전쟁광의 해골 (6th Chimera tower boss) = 전쟁광의 해골팬던트

SHOES

  • 군주의 다리 (Doomguard satyr dung) + 배란트의 액체 (1st Chimera tower boss) = 배란트의 물컹한신발
  • 배란트의 물컹한신발 (previous) + 스카디의 얼음덩어리 (5th Chimera tower boss) = 스카디얼음 부츠

LEGENDARY

  • 성화 (Guy on cliff) + 소드마스터의검 (Sephiroth from FF) = 소드마스터 계승
  • 소드마스터 계승 (previous) + 콜로서스의 돌 (1st ancient boss) + 원령골렘의 핵 (2nd ancient boss) + 제네스의 심장 (3rd ancient boss) = 고대유적의 보물
