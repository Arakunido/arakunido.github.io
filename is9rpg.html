Rank applies as an invisible ability to your current hero on LOADING. That means if you rebirth/advance in rank/change class you'll lose your current rank bonus. The only way to restore it is to load your hero again, which means you need to create new game.

You can farm rank xp in "Draenor's Forest" area, you can find a portal in the third town. (500~1200xp per Draenor)
Or you can use -rankxp command to buy rank xp with boss points(1:2 ratio).

There's a few items that you can get only via "individual drop rate" system. "Individual" means only YOU will get this item if it passes the check function. When this system triggers, it usually says something like this: "You have acquired the ITEM", then it'll spawn an item in your backpack space/area. It also uses seperate drop rate variable, you can check your current IDR in "Character Information" multiboard.
